[[[¿Cómo instalar NO INSTALAR?]]]

Extraiga el contenido (usando 7-zip) a cualquier almacenamiento persistente, para que no lo elimine más tarde.
Después de guardar el trabajo, desinstale los complementos previamente instalados, elimine todos los remanentes en el registro y el sistema de archivos.
Para una instalación instantánea basada en enlaces simbólicos *, inicie el CMD necesario del administrador (si es necesario *, corrija las rutas a VST en CMD).

* Para referencias ver: AYUDA CMD.htm

Para registrarse en el sistema, ejecute el generador de claves (Xfer_KeyGen.exe)
y seleccionando la opción de registro "Serum", haga clic delante de "TEAM R2R PRESENTS -".
Copie la clave (seleccionando y presionando Ctrl + C).

O use llaves ya hechas de:
\ Solo para información \ Si es necesario - Serum serials.txt

Abre DAW, escanea nuevos plugins y descarga Serum,
En la ventana de solicitud de clave, inserte una clave (Ctrl + V) (esto puede requerir destreza y varios intentos) 
Haga clic en Aceptar y reinicie el complemento.

Después de ingresar la clave de registro y reiniciar Serum, establezca el atributo de archivo (en propiedades)
% AppData% \ Xfer \ Serum \ user.dat
"Sólo lectura".

[[[Para referencia]]]

Posible colocación de contenido de suero:

C: \ Archivos de programa (x86) \ VstPlugins \ Xfer \ x86 \ - VST x86. El camino en tu caso puede ser diferente.
C: \ Archivos de programa (x86) \ VstPlugins \ Xfer \ x64 \ - VST x64. El camino en tu caso puede ser diferente.

C: \ Archivos de programa \ Archivos comunes \ Avid \ Audio \ Plug-Ins \ - Complementos AAX x64 para ProTools.

C: \ Users \ (Nombre de usuario actual) \ AppData \ Roaming \ Xfer \ Serum \ - registro y configuración.
C: \ Users \ (Nombre de usuario actual) \ Documents \ Xfer \ Serum Presets \ - presets y muestras.

Asignando archivos desde una carpeta:
% AppData% \ Xfer \ Serum \
presetdb.dat - base de datos de presets.
Serum.cfg: configuración y ruta a los Presets de suero.
user.dat - información de registro, clave.
