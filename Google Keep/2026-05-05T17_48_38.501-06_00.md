
Apollo Twin X Thunderbolt — PC Compatibility Summary
The Apollo Twin X Thunderbolt version requires a certified Thunderbolt 3 or 4 port. The MSI Z590-A PRO doesn't have one natively — its USB-C port is USB 3.2, which looks the same but is architecturally incompatible with the Apollo.
The board does have an internal JTBT1 header (16-pin) designed for a Thunderbolt add-on card, but here's the problem: only MSI's own ThunderboltM4 8K card is compatible with that specific connector. Cards from ASUS and Gigabyte use different pinouts and won't work. The MSI card has since been discontinued and is only sporadically available on secondary markets (eBay, MercadoLibre, AliExpress) at unpredictable prices and availability.
Options:
Source the MSI ThunderboltM4 8K on secondary market (~$1,500–2,500 MXN). Compatible but risky to find in good condition.
Switch to the Apollo Twin X USB version — plugs directly into existing USB 3.2 ports, no extra hardware needed. Cleanest solution.
Upgrade the motherboard to a Z690/Z790 with native Thunderbolt 4. Only worth it if a broader system upgrade is already planned.
Bottom line: If the Thunderbolt unit is already purchased, look for the MSI card first. If unavailable at a fair price, trading it for the USB version is the most reliable path forward.
